# Saxon Scout
## Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/Saxon-Scout-Official.git
cd Saxon-Scout-Official

# 2. Install dependencies
npm install

# 3. Run interactive setup (creates .env file)
npm run setup

# 4. Build the application
npm run build

# 5. Start the server
npm start
```

Open `http://localhost:8787` in your browser!

## Database setup

Start PostgreSQL using Docker Compose:

```powershell
docker-compose up -d
```

Apply the schema (example using psql):

```powershell
psql -h localhost -U frc -d scouting -f schema.sql
```
